Компиляция: javac FileFilter.java . Это создаст FileFilter.class и Statistics.class.
Пример запуска программы: java FileFilter -o /some/path -p result_ -f file1.txt file2.txt

1. Если директория, переданная в опции -o, не существует, то эта опция создаст новую директорию.
2. Путь, переданный в опции -o, будет автоматически будет окружен "/".
3. Ввод должен содержать хотя бы один текстовый файл.
4. Все опции должны перечислятся до файлов, иначе они не будут учитываться. 
5. Опции -o и -p могут быть переданы только один раз. 
6. Опции -s, -a, и -f могут быть переданы сколько угодно раз, но учитываться будет только первый ввод.
7. Если одновременно переданы опции -s и -f, то вызывана будет только опция -f.
8. Ненайденные файлы будут пропускаться.
